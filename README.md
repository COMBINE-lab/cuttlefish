# Cuttlefish

Building the compacted de Bruijn graph efficiently from set of reference.

## Compile
(out-of-source build)
```
$ mkdir build
$ cd build
$ cmake ..
$ make install
```

## Usage
(from project directory)
```
./bin/cuttlefish <reference-file> <k> <KMC-database-prefix> <thread-count> <output-file>
```
